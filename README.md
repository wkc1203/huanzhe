## 依赖安装
推荐使用 yarn
```
yarn
```
或者使用 npm
```
npm install
```
 
## 本地运行(以 p099 为例)
```
fis3 release dir_p099
```
num run server
## 编译打包
```
fis3 release dir_p099__prod
```
